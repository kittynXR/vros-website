# vrOS Website

Marketing site for vrOS (VR overlays + desktop control center).

Live at: https://vros.cat

## Overview

This is the official website for vrOS, highlighting its shipped features and linking to docs/support.

## Features

- Static site (no frameworks)
- Responsive layout
- Smooth animations and interactions

## Deployment

The site is deployed via Cloudflare Pages.

## Local Development

Simply open `index.html` in your browser or use a local server:

```bash
python -m http.server 8000
# Visit http://localhost:8000
```

## Domain Setup

The site uses `vros.cat` via Cloudflare:
1. Custom domain configured in Cloudflare
2. SSL/TLS mode: Full
3. Always Use HTTPS: Enabled
